Welcome to MathPath

MathPath is an adaptive mathematics learning system that identifies your strengths and weaknesses, then builds a personalized learning path.

Progress is based on mastery, not the number of questions answered. Each correct answer increases mastery and each incorrect answer decreases it.

Mastery Rules

  • Correct answer → +10% mastery
  • Wrong answer → −5% mastery
  • 90–100% mastery = topic mastered
