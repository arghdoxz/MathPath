# EduForge

**A community-driven learning platform for modules and activities with AI-assisted reviews.**

---

## Project Overview

EduForge is a student innovation project aimed at creating a low-cost, modular learning platform where users can:

- Access pre-made learning modules and activities  
- Upload their own modules for the community  
- Rate and view other users’ resources  
- Receive AI-generated reviews/summaries for uploaded modules using the OpenAI API  

The platform is designed to encourage collaborative learning and discoverability of high-quality educational content.

---

## Features

1. **Pre-made Modules** – Core learning resources provided by the platform.  
2. **User Uploads** – Users can submit their own modules (PDF, DOC, or HTML) to share with the community.  
3. **Ratings & Views** – Modules can be rated and tracked for popularity and usefulness.  
4. **AI-Assisted Reviews** – Each module can receive a summary/review generated by OpenAI’s GPT API to help learners quickly understand the content.  
5. **Search & Discover** – Find modules by topic, popularity, or AI-generated highlights.  

---

## Tech Stack

- **Frontend & Backend:** Node.js + Express  
- **Database:** PostgreSQL (hosted on Render)  
- **File Storage:** Render Object Storage or local uploads  
- **AI Integration:** OpenAI GPT API for module review generation  
- **Version Control:** GitHub
